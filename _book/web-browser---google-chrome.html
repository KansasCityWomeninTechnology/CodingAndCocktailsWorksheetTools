Web Browser: Install Google Chrome

We use Google Chrome as our web browser. Browsers often have differences in support of emerging web standards and Google Chrome is often on the leading edge of supporting these new features. In addition, Google Chrome is one of the faster browsers. You can download it here: https://www.google.com/chrome/browser/desktop/index.html
